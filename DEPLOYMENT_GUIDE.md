# Claimbase Community Edition - Quick Start

## Requirements
- Node.js 22+
- pnpm
- PostgreSQL

## Install
pnpm install

## Configure
Create a .env file in the project root with the variables required for local development.

## Database
cd lib/db
pnpm run push

## Run
Use the workspace scripts to start the API server and web portal locally.

## Notes
This community edition provides a self-hostable base for expense claim workflows.

Advanced features such as AI receipt parsing, automated reminders, advanced analytics, and richer claim-generation workflows are part of Claimbase Pro.
