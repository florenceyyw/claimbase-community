# Claimbase Community Edition

Free, open-source expense claim tracker with Telegram bot integration.

## Features
- Telegram bot for receipt uploads
- Web portal for expense management
- Manual receipt entry
- CSV export
- PostgreSQL-backed architecture
- Self-hostable

## Community Edition Scope
This repository is the community edition of Claimbase.

It is designed to provide a usable, self-hostable base for expense claim workflows while excluding advanced commercial features.

## Claimbase Pro
Claimbase Pro includes:
- AI-powered receipt parsing
- PDF and Excel claim generation
- Multi-company support
- Multi-currency workflows
- Automated reminders and scheduling
- Advanced dashboard analytics
- Bulk upload and richer claim workflows

## Quick Start
Install dependencies, configure environment variables, and run the API server and web portal locally.
